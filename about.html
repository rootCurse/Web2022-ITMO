Немного информации о моем pet проекте. Нам нужно было создать сервис для учета котов каждого пользователя. Данный сервис можно использовать в вет клинике. По пользователю можно получить всю информацию о нем и его котах Реализация содержит использование Spring Framework, а также RabbitMQ. Информация хранится в БД PostgreSQL Пример кода, реализация получение имени кота по его id и пользователю, для проверки, что кот принадлежит этому человеку 

            
public String getName(int id, int humanId) {
var cat = catsDAO.getById(id);
    if (cat.getHumanByHumansId().getHumanId() != humanId)
        return null;
    return cat.getName();
}
